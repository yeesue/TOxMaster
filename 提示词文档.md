## 五、技术规范
### 5.1 编码规范

```cpp
// 命名规范
class TestEngine;           // 类名：大驼峰
void startTest();           // 函数：小驼峰
int m_maxSpeed;             // 成员变量：m_前缀+大驼峰
const int MAX_CHANNELS = 64; // 常量：全大写+下划线

// 文件命名
TestEngine.cpp/h            // 与类名一致

// 注释规范
/**
 * @brief 启动测试
 * @param projectId 测试项目ID
 * @return 成功返回true，失败返回false
 */
bool startTest(const QString& projectId);
```

### 5.2 信号槽连接
```cpp
// 推荐使用新式连接（编译期检查）
connect(m_device, &DeviceBase::dataReceived, 
        this, &TestEngine::onDataReceived);

// 异步操作使用Qt::QueuedConnection
connect(m_device, &DeviceBase::alarmTriggered,
        this, &AlarmManager::handleAlarm,
        Qt::QueuedConnection);
```

// 使用QMutex保护共享数据
